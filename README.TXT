PROJECT TITLE: Sistema de Cadastro de Livros e Clientes (Java).
PURPOSE OF PROJECT: Desenvolver um sistema simples para praticar os conceitos de Programação 
Orientada a Objetos (POO).
VERSION or DATE: Versão 1.0 – Junho de 2025
HOW TO START THIS PROJECT: Abra o projeto no BlueJ ou qualquer IDE Java compatível.
Certifique-se de que a classe utilitária "Teclado" esteja disponível para leitura dos dados.
Execute a classe "Principal" para iniciar o programa.
AUTHORS: Vinicius Silveira
USER INSTRUCTIONS:
1. Ao executar a classe Principal, o sistema solicitará os dados de 3 livros e 3 clientes.
2. Todas as informações são inseridas via teclado (simulado pela classe Teclado).
3. O sistema calcula e exibe o preço por página de cada livro.
4. Ao final, o sistema imprime todas as informações cadastradas de forma organizada.
